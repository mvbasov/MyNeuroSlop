Давай разарботаем приложение для подготовки HTML страницы для встраивания в другую HTML страницу или Markdown документ.
Features:
1. Удалить пустые строки из всего документа. Удалять только пустые строки не меняя больше ничего. Это основной режим. Остальные функции должны включаться checkbox.
2. Удалить `<html>`, `<meta>`, `<head>`, `<body>` теги. Только теги (открывающие и закрывающие) но не их содержимое.
3. Удалить `<!DOCTYPE html>`
4. Тэг `<title>` преобразовать в Markdown заголовок.
5. Перед удалением тега запомнить его отступ и удалить именно такой отступ из его содержимого
6. Ввод через textarea или загрузка из файла.
7. Вывод копированием в клипборд или сохранением в файл.

Примерный алгоритм работы:
- Обрабатываем документ построчно.
- Если находим `<!DOCTYPE html>` или `<meta>` просто удаляем этот тег
- Если находим `<title>` добавляем его содержимое в начало выходной информации как Markdown заголовок
- Если находим `<script>` копируем исходную информацию на выход до `</script`
- Если находим `<html>`, `<head>`, `<body>` запоминаем в стеке отступ и сам тег и добавляем его к переменной хранящей текущий удаляемый отступ
- Если находим `</html>`, `</head>`, `</body>` удаляем соответствующий тег из стека и отнимаем сохранённый в стеке отступ от текущего удаляемого отступа.
- По окончании удаляем пустые строки из результата.
